# Command Whisperer

Client-side mod that displays command block commands as floating text labels. Labels are created when the command block is saved after entering or modifying a command. Note: existing command blocks will not show labels until their command is edited and saved with the mod installed.
Both Bedrock and Java players without mod will see the text too.

Includes additional button inside command block UI, that lets edit properties of displayed text.

Available settings as for now:

- Text seen through blocks Yes/No toggle
- Text render distance
- Text size
